2016年开始接触hexo+github博客。
做了一些设置，计划。

然后。。。

今天这个博客重新建立起来。

记录：

在blog这个目录下，不需要使用git status，git commit，git push等命令。

远程同步的内容只有public子文件目录里面的内容。
同步命令为hexo deploy。

错错错

千万不要hexo deploy

在主目录放一个index.html,做了修改后直接
hexo g
git commit
git push

或者访问目录直接输入https://jiaxingwangwjx.github.io/public/

修改后也许要等一下才能显示最新的文字

测试

如需测试，请先hexo clean
确保同步的不是public里面的内容

若是本地，几乎不需要任何操作。修改md文件即可。
